# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Planned

* Admin interface
* Email notifications
* Backend API
* MySQL database
* Order management dashboard

## [0.3.0] - Responsive Prototype

### Added

* Full responsive layout
* Viewport meta tag
* Mobile-first product layout
* Larger touch-friendly quantity buttons
* Fixed bottom cart button
* Safe area support for mobile devices
* Improved layout for long product names
* Dark mode support with CSS variables
* Inline SVG icons

### Changed

* Product rows now use a responsive grid layout
* App container is limited on tablet and desktop screens
* Bottom cart button no longer covers page content

## [0.2.0] - Order Review Step

### Added

* Order review page
* Customer details summary
* Product summary before final submission
* Gross total display on review page
* Phone confirmation message
* Final confirmation screen with order summary

### Changed

* Ordering flow updated to:
  Products → Cart → Customer Details → Review → Confirmation

## [0.1.0] - Initial Prototype

### Added

* Product list
* Quantity selector
* Cart calculation
* Net total calculation
* VAT calculation
* Gross total calculation
* Customer form
* Basic validation for required fields
* Confirmation message after order submission

### Notes

* This version is a static frontend prototype.
* Orders are not yet sent, stored, or emailed.
